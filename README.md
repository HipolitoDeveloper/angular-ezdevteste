# Laoyout Ecommerce
Criação de um Layout Ecommerce

## Funções
- Listagem de Itens
- Criação de Itens
- Catálogo

## Tecnologias
- [Angular] >= 11.0

## Instalação

1. Clone the repo
   ```sh
   git clone https://github.com/HipolitoDeveloper/angular-layout-ecommerce
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4.Start
   ```sh
   ng serve
   ```
